# アルパカの涙（Alpaca Tears）
アルパカの涙は、ボードゲーム「ハゲタカのえじき」と同ルールのゲームです。  
Webブラウザ上で動作するため、離れた人とも一緒に遊ぶことができます。
また、同時プレイ人数や手札数（ターン数）を可変できます。

## 必要機能
* 進行状況の共有
* チャット
* アルパカカードのシャッフル
* PWA
* Vue.js
* 各プレイヤーのカード選択確認
* プラス点のときは、一番大きいカード番号の人が獲得する
* マイナス点のときは、一番小さいカード番号の人が獲得する
* カード番号が同じ場合は次点の人が獲得する
* 特定の人と遊ぶためのルーム作成
